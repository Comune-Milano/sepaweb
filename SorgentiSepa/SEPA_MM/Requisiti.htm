Requisiti Client

SEPA@Web richiede dei requisiti tecnici minimi sia hardware che software riportati di seguito.

 

Requisiti hardware minimi

  • PC Pentium III
  • 128 Mb Ram
  • scheda video SVGA a 265 colori che funzioni ad una risoluzione minima di 800x600
  • connessione veloce ad Internet (ADSL/FIBRA) correttamente configurata.
  • in caso di accesso da un computer situato dietro un firewall o un proxy, occorre essere abilitati all'uso della porta HTTP (80) 
Requisiti software minimi
  • 1. Sistema operativo
    Microsoft Windows 98, Me, 2000 o XP (attenzione il Service Pack 2 di Windows XP richiede una configurazione ottimale del firewall personale)
  • 2. Browser Internet
    Microsoft Internet Explorer 6.0 - versione attuale 6.0 SP1 (consigliato eseguire aggiornamento automatico dal sito Microsoft mediante Windows Update)
  • 3. PDF reader
    Adobe Acrobat Reader - versione minima richiesta 5, versione attuale 7.0
Note importanti per l'installazione
  • Attenzione se sul client è installato uno dei seguenti sistemi operativi: Windows NT, Windows 2000 oppure Windows XP è necessario, per eseguire le installazioni di cui sopra, avere i diritti di administrator sul pc. In caso di dubbio contattare il Vostro responsabile informatico.
  • Abilitare il proprio browser affinchè visualizzi i popup.

Note importanti per la stampa

Affinchè le stampe, dichiarazioni e domande, siano correttamente stampate è necessario verificare, e modificare se necessario,  nel proprio browser internet, le seguneti impostazioni:

  • Dal menù File scegliere "imposta pagina". Verrà mostrata la maschera di configurazione pagina. Assicurarsi che i valori siano identici a quelli riportati di seguito.

 

 

  • Dal menù "Visualizza" scegliere la voce "Carattere". Assicurarsi che sia selezionata la dimensione "MEDIA".
  • Dal menù "Strumenti" scegliere la voce "Strumenti Internet". Dalla scheda "Generale" premere il pulsante "caratteri". Assicurarsi che nella lista "Caratteri Pagina Web" sia selezionato "Times New Roman".
